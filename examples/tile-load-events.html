--- layout: example.html title: Tile Load Events shortdesc: Example using tile load events. docs: >

Image tile sources fire events related to tile loading. You can listen for tileloadstart, tileloadend, and tileloaderror type events to monitor tile loading progress. This example registers listeners for these events and renders a tile loading progress bar at the bottom of the map.

Be ware - difference between tileloadstart and (tileloadend + tileloaderror) might reach 0 before all tiles have been loaded due to queuing. This could be the case if tile count is large or there are several layers. This shortcoming can be mitigated by querying if all map tiles have been loaded after each postrender event.
